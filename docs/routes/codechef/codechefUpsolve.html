Codechef Upsolve

To get Codechef Upsolve Contests List with Problems of a user.

URL

GET
/codechef/upsolve

Headers

Header Name Required Type Description
Authorization false string Bearer {Token}

URL Parameters

Param Name Required Type Description
handle false string Codechef handle of user (required if not authenticated).

Error Response

Reason of Error: If handle and Bearer Token is not provided. Any one is required.

Status Code: 400 BAD REQUEST

Error Response: 

            
    {
      "status": "FAILED",
      "error": "Any of handle or Bearer Token is required."
    }

Sample Call

            
With Bearer Token : 

            
  curl -X GET "https://api.codedigger.tech/atcoder/upsolve" \ 
    -H  "accept: application/json" \
    -H  "Authorization: Bearer {Access Token}"
            
          
            
Without Bearer Token : 

            
  curl -X GET "https://api.codedigger.tech/atcoder/upsolve?handle={atcoder_handle}" \ 
    -H  "accept: application/json"

Notes

User can get his upsolve contests list even if he is not logged in to our site. He has to provide his Codechef handle.
