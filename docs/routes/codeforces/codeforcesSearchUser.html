Codeforces Search User

To get the Users from the Database. Users whose Codeforces handle is similiar are returned. At max 5 users would be returned.

URL

GET
/codeforces/search

Headers

Header Name Required Type Description
Authorization false string Bearer {Token}

URL Parameters

Param Name Required Type Description
q true string Codeforces handle on which searching to be done.

Success Response

Status Code: 200

Sample Response: 

            
                {
                    "status":"OK",
                    "result":[
                       {
                          "name":"Gennady Korotkevich",
                          "handle":"tourist",
                          "rating":9999,
                          "rank":"LGM",
                          "photoUrl":"https://userpic.codeforces.org/422/title/50a270ed4a722867.jpg"
                       }
                    ]
                 }

Sample Call

            
              curl -X GET "https://api.codedigger.tech/codeforces/search?q={codeforces_handle}"

Notes

User can get the users registered with some given handle even if he is not logged in to our site. He has to provide the Codeforces handle on which searching has to be done.
