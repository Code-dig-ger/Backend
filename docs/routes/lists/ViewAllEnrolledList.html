Title

To view all the enrolled lists

URL

GET
/lists/enroll-list/

Headers

Header Name Required Type Description
Authorization true string Bearer {Token}

Success Response

All the lists that the user has been enrolled into

Status Code: 200

Sample Response: 

            
              {
                "status": "OK",
                "result": [
                  {
                    "id": 2,
                    "name": "TO-DO",
                    "description": "Problems in which you have a doubt or want to solve afterwards",
                    "slug": "to-do-1",
                    "public": true
                  },
                  {
                    "id": 4,
                    "name": "TO-DO",
                    "description": "Problems in which you have a doubt or want to solve afterwards",
                    "slug": "to-do-2",
                    "public": true
                  }
                ]
              }

Error Response

Only authorized users can enroll into a list

Status Code: 401 UNAUTHORIZED

Error Response: 

            
        {
            "status": "FAILED",
            "error": "Authentication credentials were not provided"
        }

Sample Call

Just a sample call to your endpoint in a runnable format ($.ajax call or a curl request) - this makes life easier and more predictable. 

            
              curl -X GET "https://api.codedigger.tech/lists/enroll-list/ "
